Privacy Policy

The privacy policy sets how our Smart Summary Generator(SSG) protects used information. Our Smart Summary Generator protects user information. We respects and values it’s users privacy. This privacy policy describes the information collected and recorded when the user logs in and fills up the feedback form, which will inturn help our summary generator to improve ,develop serevices and grow. If you have any more queries regarding the Privacy Policy, do no hesitate please feel free to contact us. We respect and value our user’s privacy. We don’t store any data user enters in the summary generation tool. All of our work is done in the web itself and we do not have backend server for our summarization tool. Even the weather service in our homepage is optional and does not have any effect on the summary generation and even if a user allows location permission, we don’t store the data we just feed it into an API and show the user the result. We only store data when a user submits the contact us form to know about the problem they are facing or any valuable feedback they want to give. We don’t share data with anyone and the data is protected with world class protection methods made by Google. And last of all we are open source and hosted on GitHub so anyone can see our code and find out if we are storing anymore data than we are declaring here. So now you know you can trust us so share it with anyone who may get benefit from this tool. We are waiting for your valuable feedback.
